# 澄境参赛报名帖与独立展示页设计规范

## 目标

为「澄境」生成符合学习工作赛道要求的社区报名帖，以及一个可直接上传的独立 HTML 创意展示文件。

## 报名信息

- 标签：学习工作
- 标题：【学习工作】澄境：让 AI 创作者看清设计问题的审美工作台
- 正文长度：不少于 100 字
- 必含内容：创意介绍、目标用户和痛点、价值与意义、TRAE Work 生成的 HTML 文件说明。

## 产品表达

澄境不是单纯给设计打分，而是把用户模糊的“感觉不对”转化为可理解的问题解释、可执行的修改建议和可以直接交给 AI 的 Prompt。产品闭环为：上传设计图 → AI 诊断 → 看懂问题 → 复制修改 Prompt → 查看建议效果 → 审美训练。

## 独立 HTML 页面

- 单一 HTML 文件；内嵌 CSS 和原生 JavaScript；无需构建、图片或接口。
- 主色为项目既有的蓝色与青蓝色，背景使用浅蓝渐变。
- 卡片为白色半透明 Apple Glass，包含 backdrop blur、浅色边框、柔和蓝色阴影与圆角。
- 页面内容依次为：Hero、痛点、闭环流程、四项能力、价值、页脚。
- 动效仅使用 CSS：内容淡入上移、卡片 hover 上浮；支持 prefers-reduced-motion。
- 不使用黑色主题，不使用外部 UI 框架，不伪造产品数据或用户数量。

## 验收标准

- 报名正文能直接复制发布，标签、标题、4 个必填部分完整。
- HTML 双击即可在浏览器打开，页面无外部依赖。
- 页面在桌面和窄屏下不横向溢出。
- 蓝色主调、苹果玻璃质感和产品闭环表达清晰。
